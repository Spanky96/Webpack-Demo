项目的运行与安装
``` bash
# 1. 安装node.js(npm) Git
# 2. 克隆代码至本地
git clone git@github.com:Spanky96/Webpack-Demo.git
# 3. 进入项目目录
cd Webpack-Demo
# 4. 安装依赖
npm install
# 5. 运行项目
npm run dev
# 6. 推荐使用Chrome浏览器开发和调试
http://localhost:8888

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report


```
